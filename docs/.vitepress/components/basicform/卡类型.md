## 代码演示
```
<input
    type="radio"
    :id="`${paramName}-corporate`"
    :value="options[0].value"
    v-model="selectedOption"
    @change="$emit('update:initialValue', selectedOption)"
>
<input
    type="radio"
    :id="`${paramName}-private`"
    :value="options[1].value"
    v-model="selectedOption"
    @change="$emit('update:initialValue', selectedOption)"
>
```
